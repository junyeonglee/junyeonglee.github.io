# Vim

One of the most popular editors in unix

## Mode

`i` switch to insert mode\
`esc` switch to command mode

## Work

`:set number` show line number\
`:w` write\
`:wq` write + quit\
`:q!` quit without saving

## Search

`/text` search for text (`n` for next, `N` for previous)\
`gg` fo to fist line\
`G` go to last line
